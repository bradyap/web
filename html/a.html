/a

API featuring various endpoints (info below).




/p

Pastebin backend. Takes a post request with a data string and returns a pastebin ID and URL.
("curl -d "data="Testing pastebin." -X POST https://zepr.dev/a/p")

/u

Backend for URL shortener. Accepts a post request with a URL and returns a short URL.
("curl -d "url=https://zepr.dev" -X POST https://zepr.dev/a/u")
